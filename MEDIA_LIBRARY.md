# Media Library Convention

This project now uses a normalized media structure under `assets/`:

- `assets/brand/` for brand files (logo, favicon)
- `assets/images/home/` for homepage visuals
- `assets/images/domain/` for domaine/history visuals
- `assets/images/wines/` for wine bottle visuals
- `assets/images/visits/` for visite visuals
- `assets/images/archive/` for non-production legacy/imported references
- `assets/videos/home/` for homepage videos
- `assets/documents/wines/` for downloadable wine PDF sheets
